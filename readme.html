機能概要

要素をドラッグで移動したものとして仮定した値を .data() で拾える値に書き出します。
ドラッグ中の各イベントに bind して、その値を流用することで様々な動きを実現します。

普通にドラッグしている物体を移動させても良いですし、他の要素を動かしたり、動かすための値を加工してみたり、使い方は様々です。

サンプル見たほうが早いかも。sample.html

使い方

// jQuery と jQuery UI を読み込み

～～～～

// .phantomDrag() で初期化します。

$('#drag').phantomDrag({
	xmax: 660,
	xmin: 0,
	ymax: 260,
	ymin: 0
});


// ドラッグ中に毎フレーム発生するカスタムイベントと要素の移動を .bind() で紐付けます。
// ここでは .data() で値を拾って掴んだ要素そのものの移動にあてています。

$('#drag').bind('phantomdrag-move', function (e) {

	var $el = $(e.target);

	$el.css({
		top: $el.data('phantomdrag-y'),
		left: $el.data('phantomdrag-x')
	})
});

パラメータ

  1. {number} opsions.xmax X座標の最大値
  2. {number} opsions.xmin X座標の最小値
  3. {number} opsions.ymax Y座標の最大値
  4. {number} opsions.ymin Y座標の最小値
  5. {number} opsions.delay ドラッグ追従の遅延具合。大きいほど遅く、0で遅延無し

カスタムイベント

  1. phantomdrag-start: ドラッグ開始イベント
  2. phantomdrag-move: ドラッグ中イベント（ドラッグ中に毎フレーム呼ばれます）
  3. phantomdrag-release: ドラッグ解除イベント
  4. phantomdrag-stop: ドラッグ移動停止イベント（ドラッグ解除後も続く慣性移動が停止した合図です）

.data() の値

  1. phantomdrag-x: 要素の現在のX座標
  2. phantomdrag-fx: 要素のドラッグ開始X座標
  3. phantomdrag-dx: 要素の移動先X座標
  4. phantomdrag-sx: 要素の1フレーム間でのX方向移動量
  5. phantomdrag-y: 要素の現在のY座標
  6. phantomdrag-fy: 要素のドラッグ開始Y座標
  7. phantomdrag-dy: 要素の移動先Y座標
  8. phantomdrag-sy: 要素の1フレーム間でのY方向移動量
